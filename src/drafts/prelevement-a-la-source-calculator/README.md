# BROUILLON NON TERMINÉ

Ce calculateur de prélèvement à la source n’est pas prêt pour une utilisation publique.

Le moteur personnalisé comporte encore des règles fiscales incomplètes ou non validées, notamment concernant :
- la décote ;
- le parent isolé ;
- les couples avec un conjoint sans revenu ;
- les taux individualisés ;
- certaines situations avec enfants ;
- la validation complète contre les sources officielles.

Ne pas republier cette fonctionnalité sans nouvel audit fiscal complet, correction du moteur et validation des résultats.
